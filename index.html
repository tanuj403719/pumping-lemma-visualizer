Pumping Lemma
for Regular Languages
If L is regular, there exists a pumping length p ≥ 1 such that every string s ∈ L with |s| ≥ p can be written as s = xyz satisfying:
①
|xy| ≤ p
Prefix bound
Together, x and y span at most the first p characters of s.
②
|y| ≥ 1
Non-empty pump
The middle segment y must contain at least one character — it cannot be empty.
③
∀ i ≥ 0, xyiz ∈ L
Closure under pumping
Repeating y any number of times — including zero — must keep the resulting string inside L.
Proof by Contradiction
  1. Assume L is regular.
  2. Let p be the pumping length.
  3. Pick a witness s ∈ L with |s| ≥ p.
  4. Show every valid split xyz fails condition ③ for some i.
  5. Contradiction — L is not regular. □
Segment key
x Prefix — never pumped
y Pump target — repeated i times
z Suffix — never pumped
